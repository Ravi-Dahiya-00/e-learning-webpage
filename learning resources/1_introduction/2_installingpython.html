Installing Python and Setting Up the Environment

Step 1: Download Python

  1. Go to the official Python website: python.org/downloads
  2. Select your operating system (Windows, macOS, or Linux).
  3. Click on the latest Python version and download the installer.

Step 2: Install Python

For Windows:

  1. Run the installer you downloaded.
  2. Check the box "Add Python to PATH" (important for running Python from the command line).
  3. Select "Install Now" and let it finish.

For macOS:

  1. Open the .pkg file and follow the installation wizard.
  2. Alternatively, use Homebrew:
    3. brew install python

For Linux:

  1. Use your package manager to install Python. For Debian-based systems (like Ubuntu), use:
    2. sudo apt update
    sudo apt install python3

Step 3: Verify Installation

  1. Open a terminal (or command prompt on Windows).
  2. Type the following to check if Python is installed correctly:
    3. python --version
  3. If installed, it will show the version of Python installed.

Step 4: Install a Code Editor (Optional but Recommended)

You can use any text editor to write Python code, but using an IDE or code editor makes life easier. Popular choices include:

  • VS Code: Download from code.visualstudio.com. Install the Python extension for a better coding experience.
  • PyCharm: Download from jetbrains.com/pycharm. It has built-in Python support and is great for beginners and professionals.

Step 5: Set Up a Virtual Environment (Optional but Useful)

To manage different Python projects and their dependencies, it’s a good idea to use a virtual environment.

  1. Create a virtual environment:
    2. python -m venv myenv
  2. This creates a virtual environment named myenv.
  3. Activate the environment:
    • Windows:
      • myenv\Scripts\activate
    • macOS/Linux:
      • source myenv/bin/activate
  4. Install libraries: Inside the virtual environment, install packages using pip:
    5. pip install <package_name>

You're All Set!

Now you’re all set to start coding in Python!
